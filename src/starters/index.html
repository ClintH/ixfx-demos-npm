../ starters

Sketches that have some plumbing already set up, but don't do anything themselves. Good to use as a starting point for your own exploration.

  • canvas: Canvas that fills the viewport and continually redraws based on state.
  • canvas-things: Canvas that fills the viewport and continually redraws a set of things. Useful for particle effects.
  • statemachine
  • oscillator
  • thing: demonstrates the 'thing' interaction pattern (nothing to see without adding your own code)
